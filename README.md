# 财务舞弊预测
# financial-fraud-prediction
基于机器学习的金融交易欺诈识别模型

## 项目介绍
本项目通过机器学习算法对金融交易数据进行分析与建模，自动识别高风险欺诈交易，降低金融平台风险损失。

## 技术栈
- Python
- Pandas / NumPy / Matplotlib
- Scikit-learn
- 数据预处理、特征工程、分类模型

## 数据集
数据来源于国泰安查询并下载的违规数据集

## 实现内容
- 数据清洗与异常值处理
- 样本不平衡处理
- 特征选择与特征工程
- 训练分类模型进行欺诈预测
- 模型评估（精确率、召回率、F1、AUC）

## 运行步骤
2. 运行主程序：python main.py
3. 查看训练结果与评估指标

## 项目效果
实现高准确率的金融欺诈交易识别，适用于风控、金融数据分析场景。
